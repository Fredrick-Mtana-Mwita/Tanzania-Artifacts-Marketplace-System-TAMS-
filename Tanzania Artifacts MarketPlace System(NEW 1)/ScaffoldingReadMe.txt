# Tanzania Artifacts Marketplace System

A comprehensive e-commerce platform specifically designed for buying and selling Tanzanian artifacts. This marketplace connects artifact sellers with buyers while preserving and promoting Tanzanian cultural heritage through digital commerce.

## Features

### User Roles
- **Buyers**: Browse artifacts, manage wishlist, and make purchases
- **Sellers**: List artifacts, manage inventory, and process orders
- **Administrators**: Approve sellers, manage platform, and oversee operations

### Key Functionalities
- User authentication and authorization
- Product catalog with detailed artifact descriptions and history
- Shopping cart and wishlist management
- Secure payment processing via PayPal
- Real-time notifications
- Order management system
- Seller verification process
- Admin dashboard for platform management

## Technology Stack

- **Framework**: ASP.NET Core (Razor Pages)
- **Database**: Entity Framework Core
- **Authentication**: ASP.NET Core Identity
- **Real-time Communication**: SignalR
- **Payment Integration**: PayPal API
- **Frontend**: Bootstrap, JavaScript
- **Email Service**: Built-in email service for notifications

## Prerequisites

- .NET 9.0 SDK
- Visual Studio 2022 or later
- SQL Server (Local or Express)
- PayPal Developer Account (for payment processing)
